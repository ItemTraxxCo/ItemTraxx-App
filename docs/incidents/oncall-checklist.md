# On-Call Checklist

## Start of Shift
- Check latest deployment and outstanding alerts.
- Confirm status endpoint returns healthy response.
- Confirm deployment health workflow passing.

## During Incident
- Capture evidence before restart/rollback.
- Track every action with timestamp.
- Keep one owner for execution and one for comms.

## End of Shift
- Handover unresolved items with links and exact status.
- Ensure postmortem issue is created for P1/P2 incidents.
